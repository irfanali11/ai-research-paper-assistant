# Scholar — AI Research Paper Assistant

A retrieval-augmented generation (RAG) web app that lets you upload an academic PDF, ask questions in natural language, generate a structured summary, and extract citations — all grounded in the paper's content.

Built as a portfolio project demonstrating applied AI engineering for Master's admissions and technical recruiting.

---

## Features

- **PDF Upload** — Session-scoped single-document analysis via Streamlit sidebar
- **Chat** — Ask questions; answers are retrieved from relevant chunks and generated by an LLM
- **Paper Summary** — Four-section structured output: Research Question, Methodology, Key Findings, Limitations
- **Key Citations** — Regex extraction of the References section with LLM-assisted formatting

---

## How It Works

```
PDF Upload → Text Extraction → Chunking → Local Embeddings → Chroma Vector Store
                                                                    ↓
User Question → Similarity Search → Top-K Chunks → Gemini API → Grounded Answer
```

No models are trained or fine-tuned. The pipeline uses off-the-shelf pretrained components only.

---

## Tech Stack

| Layer | Technology |
|-------|------------|
| Language | Python 3.11+ |
| Frontend | Streamlit |
| PDF parsing | pdfplumber |
| Embeddings | sentence-transformers (`all-MiniLM-L6-v2`) — runs locally, free |
| Vector database | Chroma (in-memory) |
| LLM | Google Gemini API (`gemini-2.5-flash`) — free tier |

---

## APIs Used

| API | Purpose | Cost |
|-----|---------|------|
| **Google Gemini API** | Question answering, structured summaries, citation formatting | Free tier ([get a key](https://aistudio.google.com/apikey)) |
| **sentence-transformers** | Local embedding generation | Free — no API key required |

The only external network call is to the Gemini API. Embeddings run entirely on your machine.

## Getting Started

### Prerequisites

- Python 3.11 or later
- A free [Google Gemini API key](https://aistudio.google.com/apikey)

### Installation

```bash
git clone https://github.com/irfanali11/ai-research-paper-assistant.git
cd ai-research-paper-assistant
pip install -r requirements.txt
```

### Configure Secrets

```bash
cp .streamlit/secrets.toml.example .streamlit/secrets.toml
```

Edit `.streamlit/secrets.toml`:

```toml
GEMINI_API_KEY = "your-gemini-api-key-here"
```

### Run

```bash
streamlit run app.py
```

Open **http://localhost:8501** in your browser.

---

## Usage

1. Upload a PDF research paper via the sidebar.
2. Wait for text extraction and embedding (a few seconds).
3. **Chat** — ask questions about the paper.
4. **Summary** — click *Generate Summary* for a structured overview.
5. **Citations** — click *Extract Citations* for a cleaned reference list.
